后端运行：

1. 启动 elasticsearch

在 bin 目录下运行

```bash
.\bin\elasticsearch.bat
```

2. 启动 kibana.bat （不查看 kibana 的话，可不运行）

在 bin 目录下运行

```bash
.\bin\kibana.bat
```

3. 运行启动类 MainApplication



前端运行：

```bash
## Project setup
npm install

### Compiles and hot-reloads for development
npm run serve
```
